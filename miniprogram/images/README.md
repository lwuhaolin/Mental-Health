# 图标资源说明

此目录用于存放小程序所需的图标和图片资源。

## 必需的图标文件

### 底部导航栏图标

每个图标需要两个版本（未选中和选中状态）：

1. **首页图标**
   - `home.png` - 未选中状态（灰色）
   - `home-active.png` - 选中状态（紫色）
   - 推荐尺寸: 81px × 81px

2. **AI疗愈图标**
   - `ai.png` - 未选中状态
   - `ai-active.png` - 选中状态
   - 推荐尺寸: 81px × 81px

3. **漂流瓶图标**
   - `bottle.png` - 未选中状态
   - `bottle-active.png` - 选中状态
   - 推荐尺寸: 81px × 81px

4. **个人中心图标**
   - `profile.png` - 未选中状态
   - `profile-active.png` - 选中状态
   - 推荐尺寸: 81px × 81px

### 其他图片资源

5. **默认头像**
   - `default-avatar.png`
   - 用于未设置头像的用户
   - 推荐尺寸: 200px × 200px

6. **波浪装饰（可选）**
   - `wave.png`
   - 用于首页顶部装饰
   - 推荐宽度: 750px（自适应高度）

## 图标制作建议

### 设计规范
- **颜色**: 未选中状态使用 #7A7E83，选中状态使用 #6366f1
- **风格**: 简洁、扁平化
- **格式**: PNG（支持透明背景）
- **分辨率**: 2倍图或3倍图，保证清晰度

### 在线图标资源
可以从以下网站获取免费图标：
- [iconfont](https://www.iconfont.cn/) - 阿里巴巴图标库
- [iconpark](https://iconpark.oceanengine.com/) - 字节跳动图标库
- [flaticon](https://www.flaticon.com/) - 国际图标库
- [icons8](https://icons8.com/) - 多风格图标库

### 临时方案

在图标准备好之前，小程序使用 emoji 表情作为图标，不影响功能使用。

准备好图标文件后，将其放入此目录，小程序会自动加载。

## 图标替换步骤

1. 将准备好的图标文件放入 `miniprogram/images/` 目录
2. 确保文件名与上述命名一致
3. 重新编译小程序即可看到效果

## 注意事项

- 图标文件不要过大，建议每个文件小于 50KB
- 保持设计风格一致性
- 图标要清晰可辨，避免过于复杂的设计
- 选中和未选中状态要有明显区别
